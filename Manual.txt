English:
Manual
Unzip AutoSave.zip and double-click on AutoSave.exe.
Stops cursor movement and sends CTRL+S input on active Drawing windows.
Does not work if shortcut key is not CTRL+S.

Move the folder when moving to another location. To keep everything together.

You can adjust save intervals when opening config.ini, using notepad and etc.

Running Register Start-up.exe will run AutoSave.exe on Windows Startup automatically.

Running Delete Start-up.exe will stop AutoSave.exe from running automatically on Windows startup.

Created: 07/31/2016

日本語:

使い方
お絵かき自動保存.zipを解凍してお絵かき自動保存.exeをダブルクリックするだけです。
お絵かき関連のソフトのウィンドウがアクティブかつ手が止まったタイミングでCtrl+Sを送信します。
保存のショートカットキーをCtrl+S以外にしている場合動きません。

他の場所に移動するときはフォルダごと移動してください。

config.iniをメモ帳などで開くと、自動保存間隔などを調節できます。

スタートアップに登録.exeを実行するとお絵かき自動保存ツールがWindowsを起動すると自動的に実行されるようになります。

スタートアップから削除.exeを実行するとお絵かき自動保存ツールはWindowsを起動したときに自動的に実行されなくなります。
